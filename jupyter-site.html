UX/UI Design Lead

June 2015-Current

The Project Jupyter website was my first project when I joined Project Jupyter at the beginning of the summer of 2015. The website is currently being used today and can be found here. Over the summer, I worked with project contributors and actual users of the Jupyter Notebook to find out what content the website needed to deliver. After speaking with them, I found that the website needed to provide more clarity regarding what Project Jupyter and the Jupyter Notebook are, an overview of how Project Jupyter operates as an open source project, and sections for resources and documentation for users in the open source community.
